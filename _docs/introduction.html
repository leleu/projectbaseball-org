Current Status

We are working on the data acquistion and servers. Launch is scheduled for Winter '18.

This course aims to teach anyone win their fantasy league by running their own analyses, using Python and SQL. Focus on the baseball analysis; we will provide the data and tools necessary to get started.

Programming is best learned by doing. We have removed upfront obstacles like data access or installing software so you can focus on exercises. The course is designed around these exercises, but you should be exploring on your own. For example: when we show you how to identify the players with the biggest disagreement between fWAR and bWAR, you should try a similar approach for DRS and UZR.

Objectives

  1. Analyze your fantasy league teams and trends
  2. Build your own SGP
  3. Build your own trade analyzer
  4. Build your own player projection system, like ZiPS or PECOTA
  5. Foundational knowledge to perform your own player or team analysis

Who is this for?

  • Bloggers and data journalists who want to write about the modern game
  • Fantasy players who want to run their own Moneyball analysis on their fantasy leagues
  • Baseball geeks who want a centralized place to access baseball data
