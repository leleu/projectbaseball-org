Introduction

This course aims to teach anyone how to analyze baseball data, using Python and SQL. We have prepared the dataset for you, and will provide super easy tools to connect to your fantasy league.

Stay Informed

Sign up for the mailing list if you want to be notified when the modules get posted.

Objectives

  1. Build data analysis skills sufficient to provide a competitive edge in your fantasy league.
  2. Provide the tooling and data needed to run your own data analysis.
  3. Teach by example, using live everything.
