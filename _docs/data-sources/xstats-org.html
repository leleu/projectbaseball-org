--- title: xStats.org (in progress) permalink: /docs/xstats-org/ ---

xStats.org is a data source that uses exit velocity, launch angle, and other inputs to estimate the results of similarly struck balls.

E.g., if a batter hits the ball with a 98mph exit velocity and a 42% launch angle to left-left-center, how often is that a single/double/triple/homer/out, across all the batted balls in the game.

Expected to launch September 23, 2018.
